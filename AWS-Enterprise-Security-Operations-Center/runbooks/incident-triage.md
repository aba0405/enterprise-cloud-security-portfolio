# Security Incident Triage Runbook

## Severity Classification

### Critical

Examples:

* Active compromise
* Command and control activity
* Credential theft
* Data exfiltration

Response Time: Immediate

---

### High

Examples:

* Privilege escalation
* Unauthorized access attempts

Response Time: 1 Hour

---

### Medium

Examples:

* Misconfigurations
* Vulnerability findings

Response Time: Same Business Day

---

### Low

Examples:

* Informational alerts
* Compliance findings

Response Time: Scheduled Review

---

## Triage Workflow

1. Validate alert
2. Determine business impact
3. Identify affected resources
4. Assign severity
5. Initiate containment if required
6. Document findings
