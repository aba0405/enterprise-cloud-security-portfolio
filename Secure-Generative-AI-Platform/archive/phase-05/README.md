# Phase 05 - Amazon Bedrock Integration

## Objective

Integrate Amazon Bedrock with AWS Lambda to enable AI-generated responses for the Secure Generative AI Platform. This phase replaces the placeholder Lambda response with dynamic responses generated by Anthropic Claude running on Amazon Bedrock.

---

## AWS Services Used

* Amazon Bedrock
* AWS Lambda
* Amazon API Gateway
* AWS IAM
